# 🧠 GenAI 활용법

## 1. 대화 형태에 따른 활용

* **싱글턴(Single-turn)**:
  → 단일 명령 중심
  예: "요약해줘." → 끝
* **멀티턴(Multi-turn)**:
  → 연속적인 상호작용
  예:

  * "요약해줘."
  * "시사점도 뽑아줘."
  * "추가로 시각화 해줄 수 있어?"

## 2. 역질문 받기

* AI에게 거꾸로 질문 받도록 요청 → 사고 전환 및 학습 강화에 유용
  예:

  * "이걸 바탕으로 나에게 질문해줘."
  * "내 주장에 반박해봐."

## 3. 롤플레이 활용

* AI에게 특정 역할을 부여해 시뮬레이션 진행
  예:

  * "너는 면접관이야. 나는 지원자야."
  * "너는 클라이언트, 나는 기획자야."

# 💡 좋은 프롬프트 구조

* **기본 3단 구성**:
  `역할(Role) + 배경(Context) + 작업지시(Task)`

* **주의사항**:

  * **TMI 금지**: 불필요한 배경은 오히려 혼란만 가중
  * **전문 용어 설명 생략주의**:
    AI는 업계의 은어/약어를 잘못 해석할 수 있음 → 가능한 풀어서 명시
