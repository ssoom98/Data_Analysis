## 🧠 LLM vs SLM: 정의와 비교

| 항목        | LLM (Large Language Model)             | SLM (Small/Specific Language Model)           |
| --------- | -------------------------------------- | --------------------------------------------- |
| 🔍 약어 의미  | **Large** Language Model               | **Small** or **Specialized** Language Model   |
| 📚 학습 범위  | 광범위한 범용 데이터 (인터넷 전반, 백과사전 등)           | 특정 도메인 또는 업무 영역 중심 (ex. 의료, 법률, 고객상담 등)       |
| 🧠 모델 크기  | 수십억\~수천억 파라미터 (예: GPT, Claude, Gemini) | 수백만\~수억 파라미터 수준                               |
| 🛠️ 사용 목적 | 범용 언어 처리 (요약, 번역, 질문응답 등 전반적 활용)       | **특정 업무/도메인에 최적화**된 태스크 수행                    |
| ⚙️ 장점     | 높은 성능, 범용성                             | 빠른 응답속도, 저사양 실행 가능, 높은 정확도(특정 분야 한정)          |
| 🧱 예시     | GPT-4, PaLM, Claude, LLaMA             | Med-PaLM (의료), FinGPT (금융), 너가 만든 체중예측 모델도 가능 |

---

## 💡 예시로 이해하기

* 📌 **LLM**:
  “전 세계의 지식을 두루 알고 있는 박사님. 모든 질문에 답할 수 있으나, 특정 세부 지식은 다소 모호할 수 있음.”

* 📌 **SLM**:
  “한 분야에 엄청나게 특화된 전문가 (ex. 암 전문 의사, 세무회계사). 범용 지식은 약하지만 자신 분야에선 최강.”

---

## ✅ 요약

> **SLM은 특정 업무/도메인에 특화된 임베디드 LLM이다.**
> 작고 가볍고 빠르며, 특화된 태스크에 **LLM보다 훨씬 정확할 수 있다.**
